# LatihanUKLNo3

Program ini digunakan untuk menghitung faktorial dari sebuah bilangan bulat positif yang dimasukkan oleh pengguna.

Program diawali dengan menulis Scanner yang memungkinkan pengguna memasukkan data dari keyboard. Kemudian, program menampilkan pesan untuk meminta pengguna memasukkan sebuah bilangan bulat positif setelah itu nilai yang dimasukkan pengguna disimpan dalam variabel yang telah di tentukan. Program selanjutnya menggunakan perulangan untuk menghitung hasil faktorial dengan cara mengalikan angka dari 1 hingga bilangan yang dimasukkan. Setelah perhitungan selesai, program menampilkan hasil faktorial tersebut ke layar.
